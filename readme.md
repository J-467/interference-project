## Running the project


### Dependencies
Install pip if you don't have it
```
python get-pip.py
```

Install Python dependencies (using the terminal):

 ``` bash
1. pip install pymongo
2. pip install spectrum
3. pip install pandas
```
*Spectrum also installs numpy and matplotlib*
